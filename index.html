Aeltere Version installieren?

Version

Beim Downgrade koennen Datenmigrationen neuerer Versionen fehlen. Dadurch koennen Notizbuecher, Einstellungen oder Dateiformate inkompatibel werden.

Bitte erstelle vor dem Fortfahren ein Backup deiner aktuellen Daten und installiere nur, wenn du den Downgrade bewusst moechtest.

Hinweis: Falls danach in der App ein Hinweis auf eine neuere Version erscheint, kannst du diesen einfach ignorieren.
